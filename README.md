1.支持字体样式，字体颜色，字体阴影，对齐方向，行间距。

2.支持复制粘贴剪贴，格式化，撤销重做。

3.支持形状绘制。

4.连接线（部分完成，完善中）

5.位置，组合，对齐

6.元素翻转，旋转。

7.填充颜色，支持线性渐变色，径向渐变色等

8.支持箭头样式（部分完成，完善中）

9.锁定与解锁

10.快速样式

11.支持矢量文本，二维码

12.支持插入图片，视频，SVG

13.支持画板大小，方向，标尺，网格是否显示，画板背景色

14.支持流程图（在文件新建下-基本绘图-流程图）
